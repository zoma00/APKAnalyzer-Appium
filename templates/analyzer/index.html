Application purpose

This application serves as an Android APK Analysis and Testing Platform.It allows users to:

1. Upload and Manage APKs: Users can upload their Android app installation files (APKs), view details, and manage (update/delete) their uploaded apps.

2. Automated Appium Testing:

  • The platform leverages Appium to automatically test uploaded APKs on an Android emulator.

      • It performs actions like:

    • Installing the app.
    • Capturing UI hierarchy.
    • Simulating button clicks.
    • Checking for screen changes.
    • Recording screenshots and videos.
    • Storing test results (UI hierarchy, screen changes) in the database.

      3.Accessibility and Multilingual Support:** The application aims to be inclusive by offering:

    • Accessibility features (font size adjustment, high contrast mode).
    • Bilingual support (English and French) to cater to a wider audience.

    Essentially, this platform streamlines the process of APK testing and analysis, making it easier for developers to assess their apps' functionality, accessibility, and user experience across different languages.

    Additional Notes:

    * The use of Docker ensures easy deployment and environment consistency.
    * Django provides a robust backend framework for handling user authentication, data management, and web interface generation.
    * MySQL serves as the database for storing app information and test results.
    * Django templates, HTML, CSS, and JavaScript are used for the frontend presentation.
    * Appium handles the automated testing on Android emulators.
    * Accessibility features are implemented using CSS and JavaScript.
    * Django's internationalization framework facilitates multilingual support.
    * Django unit tests contribute to code quality and maintainability. The challenge emphasizes a full-stack development approach, combining backend, frontend, testing, and accessibility considerations.
